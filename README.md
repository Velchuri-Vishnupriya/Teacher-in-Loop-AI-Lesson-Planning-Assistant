The Teacher-in-the-Loop AI Lesson Planning Assistant is an AI-powered educational support system developed as part of the IIT Jodhpur Internship Program.

The system combines teacher expertise with Generative AI to create high-quality lesson plans through a collaborative workflow. Instead of allowing AI to independently generate lesson plans, teachers remain actively involved throughout the planning process by providing inputs, answering clarification questions, reviewing generated plans, and refining them iteratively.

The application follows a Human-AI Collaboration approach and generates lesson plans using the 5E Instructional Model.
